## 🐛 问题排查

### 问题: 点击页码后商品不变

**检查步骤**:
1. 打开浏览器开发者工具（F12）
2. 切换到 Console 标签
3. 点击页码按钮
4. 查看是否有错误信息

**可能原因**:
- JavaScript 文件未正确加载
- shop.js 未更新
- 浏览器缓存

**解决方法**:
```
1. 硬刷新: Ctrl + Shift + R (Windows)
2. 清空缓存: Ctrl + Shift + Delete
3. 检查 shop.js 文件是否已保存
```

---

### 问题: 页面刷新但显示错误的商品

**检查**:
- 查看控制台是否有错误
- 检查 `shouldResetPage` 标志是否正确设置

**调试代码**:
```javascript
// 在 shop.js 的 render() 方法中添加日志
render(products) {
    console.log('shouldResetPage:', this.shouldResetPage);
    console.log('currentPage:', this.paging.getCurrentPage());
    // ...
}
```

---

### 问题: 过滤后没有跳到第1页

**检查**:
- toolbar 回调是否设置了 `shouldResetPage = true`
- render() 方法是否检查了标志
