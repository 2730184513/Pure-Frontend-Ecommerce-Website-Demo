## 🔍 常见问题排查

### 问题 1: items-per-page 点击没反应
**检查**:
- [ ] 浏览器控制台有无 JavaScript 错误？
- [ ] `show-dropdown` 元素是否存在？
- [ ] CSS 中 `.custom-dropdown.open` 样式是否生效？

**解决**:
- 刷新页面（Ctrl + F5）
- 清除浏览器缓存

---

### 问题 2: 首页产品不显示
**检查**:
- [ ] 控制台是否显示 "🛍️ Initializing generic product repository..."？
- [ ] 是否有数据加载错误？
- [ ] `.product-grid` 元素是否存在？

**解决**:
- 打开控制台，查看网络请求
- 确认 JSON 文件加载成功
- 检查 `product-repository.js` 是否正确加载

---

### 问题 3: Price 滑块还是 50000
**检查**:
- [ ] `filter-sidebar.js` 是否已更新？
- [ ] 浏览器缓存是否清除？

**解决**:
- 硬刷新：Ctrl + Shift + R (Chrome/Firefox)
- 检查 `filter-sidebar.js` 第 57 行和第 134 行
